Dream-E Documentation

Visual node editor for interactive fiction and open-world text-adventure RPGs.

Dream-E is a browser-based authoring tool for creating branching narrative games. It provides a visual node graph editor for designing story structure, an entity system for world-building, and an AI-powered Open World mode that generates scenes dynamically based on player actions.

Tech stack:

React 18 TypeScript Vite Zustand React Flow Tailwind CSS Dexie.js (IndexedDB) Howler.js Lucide Icons
