Architecture & File Structure

High-Level Architecture

Browser (React SPA)
  ├── Dashboard (project list, create/import/delete)
  ├── Editor (React Flow canvas + inspector + toolbar)
  │     ├── Node editing (scene/choice/modifier/comment)
  │     ├── Entity management (characters, locations, objects, concepts)
  │     ├── Variable management (game stats, inventory)
  │     ├── AI Chat (agentic loop via Claude Code)
  │     └── AI Settings (image gen, TTS, writer config)
  └── Player (AdventureEngine)
        ├── Standard mode (predefined branching narrative)
        └── Open World mode (AI-generated scenes)
              ├── Context construction (openWorldContext.ts)
              ├── AI generation (openWorldService.ts → vite middleware → Gemini/OpenAI API)
              ├── Image generation (Gemini/BFL/OpenAI)
              ├── Music search (BM25 FastAPI server)
              └── TTS (Gemini chunked streaming)

Vite Dev Server (vite.config.ts middleware)
  ├── POST /api/open-world     → Gemini / OpenAI streaming
  ├── POST /api/generate-image → BFL / OpenAI / Gemini image gen
  ├── POST /api/generate-tts   → Gemini TTS
  ├── POST /api/chat           → Claude Code subprocess
  ├── POST /api/chat-reset     → Clear chat history
  └── GET  /api/music/*        → Proxy to BM25 server (port 7862)

Persistence
  ├── IndexedDB (via Dexie.js) → Projects, assets, save slots
  └── localStorage → User preferences, AI settings, save data

Data Flow Patterns

Editing Flow

User edits field in Inspector
  → Inspector calls store.updateNode(id, { data: { ...changes } })
  → useProjectStore.set() via Immer (immutable mutation)
  → recordHistory() queued (debounced 500ms for text, immediate for structural)
  → isDirty = true
  → scheduleAutoSave() triggers after 2s
  → saveProject() writes to IndexedDB (rehydrates blob URLs to base64)
  → Editor.tsx sync trigger updates React Flow nodes/edges arrays
  → Canvas re-renders (skipped if user is typing in text input)

Open World Generation Flow

Player types action → handleChoice()
  → useProjectStore.getState().currentProject (fresh, not stale player store copy)
  → buildOpenWorldContext(project, session, action)
  → POST /api/open-world (SSE streaming)
  → parseOpenWorldResponse(fullText)
  → onComplete(result) → display pending scene text + choices
  → Background: generateImage() + searchAndAssignMusic()
  → Player clicks "Continue to Next Scene"
  → handleAdvanceToOwScene()
    → Create new scene node + edge in project graph
    → Deferred: apply entity updates (profilePatch, stateHistory, stateNote)
    → Deferred: apply variable changes
    → saveAndOffloadAssets() (base64 → blob URLs)

AI Chat (Agentic Loop) Flow

User sends message in ChatWindow
  → buildUserMessage(text) includes game state context
  → POST /api/chat (Claude Code subprocess)
  → Parse response for <<<SW_CMD>>> blocks
  → Execute each command via gameStateAPI.executeCommand()
  → buildResultsMessage(results) fed back to agent
  → Loop until no more commands or MAX_ITERATIONS (10)
  → Return final text to ChatWindow
