MODERN THEME ASSETS
===================

The modern theme primarily uses CSS effects (glassmorphism)
and doesn't require many image assets.

OPTIONAL IMAGES:
- pattern.png       - Subtle background pattern
- noise.png         - Noise texture overlay

OPTIONAL SOUNDS:
- click.mp3         - Soft button click
- transition.mp3    - Smooth transition sound

STYLE NOTES:
- Clean, minimal design
- Lots of blur effects (backdrop-filter)
- Rounded corners (16px radius)
- Semi-transparent backgrounds
- Sans-serif fonts (Inter)
- Purple/indigo color accents
